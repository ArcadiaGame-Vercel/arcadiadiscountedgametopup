# 🎮 Arcadia Game Top-Up Website

A simple **Codashop-style game top-up website** built with Next.js and deployed on Vercel.
This project allows users to enter their game details, select diamond packages, and submit orders for manual processing.

---

## 🚀 Features

* 🎯 User ID & Server ID input
* 💎 Diamond package selection
* 📦 Order summary preview
* 💳 Manual payment (GCash ready)
* ⚡ Fast deployment via Vercel

---

## 🛠 Tech Stack

* Next.js
* React
* Vercel (Deployment)

---

## 📦 Installation (Optional - Local Setup)

```bash
npm install
npm run dev
```

Open: http://localhost:3000

---

## 🌐 Live Deployment

This project is deployed using Vercel.
You can easily connect your GitHub repository and deploy in one click.

---

## 💸 How It Works

1. User enters **Game UID & Server ID**
2. Selects a **diamond package**
3. Confirms the order
4. Pays via **GCash**
5. Admin manually processes the top-up
